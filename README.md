# url-shortener
A simple app used to shorten URLs

## To run project:

In the terminal, do the following;

- `git clone git@github.com:fogha/url-shortener.git`
- `cd url-shortener`
- `npm run start`

## To run tests on the project:

- Run `yarn run test`
